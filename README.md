# floating-datasets

FloW-BEV: Reconstructed from the FloW-RI dataset, it provides 3D annotations for its 2,000 frames of image and point cloud data. You can download from： https://pan.quark.cn/s/5f0f1912a0a3

WaterScenes-BEV: Reconstructed from the WaterScenes dataset, it provides 3D an-notations for a subset of its extensive multimodal data collection.  You can download from：https://pan.quark.cn/s/df1ebc23f56c
